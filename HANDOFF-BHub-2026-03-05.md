# HANDOFF — B Hub
*Last updated: March 5, 2026 ~12:00 PM ET*

## Project Overview
B Hub is the homepage and app switcher portal for the B-Suite ecosystem. Static HTML site that serves as the entry point to all B-Suite apps. Also hosts sub-pages for B Resources (Library + Vault coming-soon pages) and formerly hosted B Marketing's sub-menu (now spun out to its own app).

## Tech Stack
- **Frontend:** Static HTML + CSS (no framework, no build step)
- **Hosting:** Vercel — auto-deploys from `brhecht/bhub` on GitHub
- **Design:** Warm cream (#FAF6F1), Playfair Display + DM Sans, pastel gradient cards

## Design Decisions & Constraints
- **Static HTML, no framework** — bhub is a simple portal. No state, no API calls, no build step. Just links.
- **Both B Marketing and B Resources spun out** — Both are now standalone React apps. bhub links to their Vercel URLs.
- **Sub-menu pattern** — B Marketing and B Resources use the same vertical list layout: white card rows on a themed background, status badges, icons. This is now the standard B-Suite "sub-hub" design.
