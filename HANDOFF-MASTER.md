# B-Suite Master Handoff — MOVED

This file used to be the B-Suite cross-app master index. As of **May 25, 2026**, it was migrated to a dedicated private repo to stop leaking the entire B-Suite architecture, device list, deploy paths, and cross-project context to the public web.

**Current master handoff:** [`brhecht/bsuite-handoffs/HANDOFF-MASTER.md`](https://github.com/brhecht/bsuite-handoffs/blob/main/HANDOFF-MASTER.md) (private — auth required).

**Per-app handoffs:** Now live at `brhecht/bsuite-handoffs/<app>/HANDOFF.md`.

**Why this stub exists:** Older skill versions and bsync builds still expect to find HANDOFF-MASTER.md in the bhub repo. This stub points them to the new location.

**For Claude:** Pull `brhecht/bsuite-handoffs` (private — requires PAT) and read `HANDOFF-MASTER.md` from there.
